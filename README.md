PROGETTO DI VOTO ELETTRONICO

1. Installare la dipendenza:
   py -m pip install -r requirements.txt
2. Avviare:
   py main.py

SELF-TEST
   py main.py --self-test

In caso di test automatico eseguire demo inserndo e inviando numero 5

Compariranno il risultato più i dati della votazione seguiti dagli esiti dei vari test di verifica universale

In caso di test manuale avvalersi delle credenziali demo:

premere tasto 6 per ottenere credenziali demo di votanti

premere tasto uno per iniziare sessione di autenticazione ed in caso di credenziali corrette espressione del voto

ripetere operazione per più votanti (opzionale)

premere tasto 4 per verificare la proprià ricevuta (NOTA nel campo Nome file ricevuta precedere il nome del file da data/)

premere tasto 3 per accedere all'area dell'autorità elettorale tramite credenziali ottenibili in mostra credenziali demo (tasto 6)

autenticarsi con tali credenziali

In menu AE:

premere tasto 1 per ottenere stato delle votazioni

premere tasto 2 per chiudere le urne (verifica temporale richiesta. In questo codice la data di fine è impostata a 31/12/2026 quindi non è possibile chiudere le urne legalmente)

premere tasto 3 per chiudere urne a scopo dimostrativo

premere tasto 4 per far partire lo scrutinio

inserire le credenziali dei trustee ottenute mostrando le credenziali demo di almeno 3 trustee per far partire lo scrutinio

premere tasto 5 per far partire i test di verifica universale

premere tasto 6 per tornare a menu generale

Out menu AE

premere tasto 7 per terminare il codice
